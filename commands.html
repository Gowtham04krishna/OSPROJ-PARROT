List of System Commands and Their Descriptions

Command Description 
uname -a
 Displays system information such as the kernel name, version, release number, machine architecture, and operating system. 
hostnamectl
 Allows you to query and change system hostname settings. Provides information about the system's hostname, as well as the static and transient hostname settings. 
ifconfig
 Displays information about all network interfaces currently active on the system, including their IP addresses, MAC addresses, and other network-related details. 
iwconfig
 Configures wireless network interfaces. Displays detailed information about wireless interfaces, such as their wireless settings, signal strength, and other parameters. 
arp -a
 Displays and manipulates the Address Resolution Protocol (ARP) cache. Shows the mapping between IP addresses and MAC addresses on the local network. 
netstat -tuln
 Displays network-related information such as active network connections, routing tables, interface statistics, masquerade connections, and multicast memberships. 
nslookup google.com
 Queries DNS servers to retrieve domain name system (DNS) information, including the IP address associated with a domain name. 
nmap 192.168.1.6
 Nmap is a powerful network scanning tool used to discover hosts and services on a computer network. Scans the specified IP address to identify open ports and services running on the target machine. 
ps aux
 Displays information about active processes running on the system. Provides a snapshot of the currently running processes, including their process IDs (PIDs), resource usage, and other details. 
nmcli
 A command-line interface for NetworkManager. Allows users to control and monitor network interfaces, connections, and devices. 
journalctl -n 10
 Queries and displays logs from the systemd journal. Allows users to view system logs, filter log entries based on various criteria, and monitor system activity. 
lsmod
 Displays information about loaded kernel modules. Lists all currently loaded modules along with information about their dependencies and usage count. 
uptime
 Displays the system's uptime, which indicates how long the system has been running since it was last booted. 
last -n 10
 Displays a list of recent login sessions on the system, including information about when users logged in and logged out, as well as system reboots. 
top -b -n 1
 Displays real-time information about system resource usage, including CPU usage, memory usage, and process statistics. Provides an interactive interface for monitoring system performance. 
free -h
 Displays information about system memory usage, including total memory, used memory, free memory, and memory buffers/cache. 
df -h
 Displays information about disk space usage on the system. Shows the amount of disk space used and available on each mounted filesystem. 
anonsurf status
 Used to anonymize the system's network traffic by routing it through the Tor network, enhancing privacy and anonymity while browsing the internet. Specific to Parrot OS. 
date
 Displays the current date and time according to the system's clock settings. Can also be used to set the system's date and time. 
passwd
 Used to change the password for the current user or for another user account on the system. Prompts the user to enter their current password and then to enter and confirm a new password. 
scp
 Securely copies files and directories between hosts on a network using Secure Copy Protocol (SCP) to transfer data over SSH (Secure Shell) connections. 
ssh
 Establishes secure shell connections to remote hosts over a network, providing encrypted communication between the local and remote systems, allowing users to execute commands on the remote host or transfer files securely. 
chmod
 Changes the permissions of files and directories in the filesystem, allowing users to specify who can read, write, and execute the file or directory. 
chown
 Changes the ownership of files and directories in the filesystem, allowing users to change the user and group ownership of a file or directory to a specified user and group. 
find
 Searches for files and directories in the filesystem based on various criteria such as file name, size, type, and permissions. A powerful tool for locating files and performing operations on them.
